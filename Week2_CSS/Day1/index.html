What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout HTML elements — for example, changing colors, fonts, spacing, and positioning.

1. Inline CSS

Written directly on the HTML tag using the style="" attribute.

This paragraph uses INLINE CSS — color and font-size written on the tag.

This paragraph also uses INLINE CSS — background, padding, border-left.

2. External CSS

Written in a separate .css file and linked in the <head> using <link rel="stylesheet" href="day1.css" />. The header, nav, and this section are all styled using external CSS.
