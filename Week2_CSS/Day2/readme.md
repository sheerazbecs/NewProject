Day 2: Colors, Fonts & Text Styling
Key Properties Learned

1. Color Properties
color: #8b3a1e;           /* Text color - rust/brown */
color: #c9993a;           /* Text color - gold */
background-color: #f0e6d3; /* Background color - cream */

2. Font Properties
font-family: 'Playfair Display', serif;    /* Elegant font */
font-family: Georgia, serif;               /* Classic font */
font-size: 17px;                           /* Text size */
font-weight: bold;                          /* Bold text */
font-style: italic;                          /* Italic text */

3. Text Properties
text-align: center;        /* Center alignment */
text-transform: uppercase;  /* ALL CAPS */
letter-spacing: 4px;        /* Space between letters */
line-height: 2.5;           /* Space between lines */
text-decoration: none;      /* Remove underline */
