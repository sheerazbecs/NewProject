Examples of Common Tags:
Tag	Purpose	Example
<h1>	Main heading	<h1>Title</h1>
<p>	    Paragraph	    <p>Text here</p>
<a>	    Link	        <a href="link.html">Click</a>
<img>	Image	        <img src="photo.jpg">

Remember:
Headings go from largest (h1) to smallest (h6)
Paragraphs separate text blocks
Links need href attribute
Images need src (image file) and alt (description)
