LISTS
Purpose:
Organize related items in bullet points or numbered format.

1. Unordered List (<ul>)
Creates bullet points
Use when order doesn't matter
Example: Shopping list, features, categories

2. Ordered List (<ol>)
Creates numbered list
Use for steps, rankings, sequences
Example: Instructions, rankings, procedures

3. List Item (<li>)
Each item in any list
Must be inside <ul> or <ol>
--------------------------------------
TABLES
Purpose:
Display data in rows and columns (grid format).

Table Structure:
<table> - Container for entire table
<tr> - Table Row (horizontal)
<th> - Table Header (bold, centered)
<td> - Table Data (normal cell)

Table Sections:
<thead> - Header section (column titles)
<tbody> - Main data rows
<tfoot> - Footer (totals, summary)
--------------------------------------
Rules to Remember:

1. <ul> = Unordered List (bullets)
2. <ol> = Ordered List (numbers)
3. <li> = List Item (goes inside ul or ol)
4. Always close tags properly
5. One <li> per item

Key Points:
<table> = Whole table
<tr> = Each row (horizontal line)
<th> = Header cell (bold, for titles)
<td> = Data cell (normal content)
