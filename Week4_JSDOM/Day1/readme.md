Day 1 - DOM Introduction
What is DOM?
DOM stands for Document Object Model. It represents the HTML document as a tree structure that JavaScript can interact with.

Basic Concepts Covered
1. Selecting Elements
document.getElementById('demo')
This finds an HTML element by its ID.

2. Changing Content
element.textContent = 'New text';
Changes the text inside an element.

3. Changing Styles
element.style.color = 'blue';
Changes CSS properties using JavaScript.

4. Show/Hide Elements
element.style.display = 'none';  // Hide
element.style.display = 'block'; // Show
How to Run
Open index.html in your browser
Click the buttons to see DOM manipulation in action
Simple Summary
DOM lets JavaScript talk to HTML
We can change text, colors, and visibility
document.getElementById() is used to find elements
