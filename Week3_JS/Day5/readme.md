This file combines everything from Day 1 to Day 4 using an Employee & Salary theme.

It stores employee details using **variables** and prints them with a template literal. A **condition** checks if the employee qualifies for a senior role. Finally, an array of employee objects is looped through and the salary function is called for each one to print the full result.
